# Supermarket Sales Analysis

## Project Overview
This project analyzes supermarket sales data using Python, Pandas, and Matplotlib.

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib

## Analysis Performed
- Total Sales Analysis
- Sales by Branch
- Product Sales Analysis
- Payment Method Analysis
- Customer Rating Analysis

## Key Findings
- Compared branch performance
- Identified top-selling products
- Analyzed payment preferences
- Visualized sales trends using charts
